# О VKUI

**VKUI** — это библиотека адаптивных React-компонентов для создания веб-приложений. Библиотека основана на [дизайн-системе ВКонтакте](https://www.figma.com/@vk) и реализует её интерфейсы для различных платформ.

Начните разрабатывать красивые интерфейсы уже сейчас, изучив инструкцию по [быстрому старту](https://vkcom.github.io/VKUI/#/QuickStart).

![VKUI logo on dark background](/img/intro-banner.png)

## Наши преимущества

- Адаптивность.\
  Библиотека подстраивается под экраны разного размера, делая интерфейс доступным на смартфонах, планшетах и компьютерах.

- Мультиплатформенность.\
  Интерфейсы на VKUI практически неотличимы от дизайна платформ, внутри которых они открыты. Библиотека умеет "мимикрировать" под наши iOS и Android клиенты, под дизайн [vk.com](https://vk.com) и [m.vk.com](https://m.vk.com).

- Разные цветовые схемы.\
  Тёмная тема — один из самых важных факторов доступности.
