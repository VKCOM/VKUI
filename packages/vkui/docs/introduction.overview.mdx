## Песочница VKUI

Данная страницы в первую очередь предназначена для разработки. Здесь можно протестировать компоненты более детально переключая
локальные и/или глобальные свойства.

Информация об установке и использовании библиотеки, про описание компонентов, про особенности – представлена в основной документации
[vkui.io](https://vkui.io).

### Навигация

В меню (на настольных экранах слева, на мобильных снизу) находится основная навигация по компонентам. Для более наглядной навигации
собрали все компоненты в [витрине](?path=/docs/components-overview--docs).

### Настройки

На странице каждого компонента есть два набора инструментов:

- **Toolbar** (располагается сверху) – предоставляет глобальные настройки, вот основные:
  - переключение размера области видимости;
  - переключение темы как локально, так и глобально;
  - ссылка на исходник в **GitHub**;
  - ссылка на основную документацию в [vkui.io](https://vkui.io);
  - переключение платформы;
  - переключение направление текста;
  - открытие страницы с компонентом в новом окне;
  - копирование ссылки на страницу.
- **Addon Panel** (располагается снизу) – предоставляет несколько вкладок относящиеся непосредственно к компоненту. Основные:
  - вкладка **Controls** с переключением свойств компонента;
  - вкладка **Source** с представлением примера на странице в JSX;
  - вкладка **Accessibility** с рекомендациями по доступности;
  - вкладка **Design** с превью дизайна компонента в **Figma**.

### FAQ

#### Не вижу **Toolbar** / **Addon Panel**

Это значит, что на данный момент они у вас отключены в настройках песочницы. Нажмите на ⚙️ в меню и убедитесь, что включены
пункты **Show toolbar** и **Show addons panel**.
