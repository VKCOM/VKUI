# @vkontakte/vkui-mcp

MCP-сервер для документации VKUI: даёт AI-ассистентам доступ к компонентам, хукам, примерам кода и рекомендациям по миграции.

## Документация

Подробную документацию по пакету вы можете найти на странице [vkui.io/mcp](/mcp).

Там находятся:

- инструкции по подключению в разные MCP-клиенты;
- переменные окружения и поддерживаемые версии;
- список ресурсов и инструментов с описаниями и примерами.

## Для разработчиков пакета

- **Тесты:** `yarn test` (юнит- и интеграционные тесты с моком HTTP).
- **Сборка:** `yarn build` (результат в `dist/`).
- **Публикация:** `prepack` запускает `yarn run build`; в npm-пакет включается только `dist/` (см. `files` в `package.json`).
