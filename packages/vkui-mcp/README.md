# @vkontakte/vkui-mcp

MCP-сервер для документации VKUI: даёт AI-ассистентам доступ к компонентам, хукам, примерам кода и рекомендациям по миграции.

## Документация

Подробную документацию по пакету вы можете найти на странице https://vkui.io/overview/mcp/.

Там находятся:

- инструкции по подключению в разные MCP-клиенты;
- переменные окружения и поддерживаемые версии;
- список ресурсов и инструментов с описаниями и примерами.

## Для разработчиков пакета

- **Тесты:** `node --run test` (юнит- и интеграционные тесты с моком HTTP).
- **Сборка:** `node --run build` (результат в `dist/`).
- **Публикация:** `prepack` запускает `node --run build`; в npm-пакет включается только `dist/` (см. `files` в `package.json`).
