# @vkontakte/storybook-addons

Набор аддонов для Storybook, используемых в экосистеме VKUI.

## Аддоны

| Аддон                | Описание                                            | Документация                                    |
|----------------------|-----------------------------------------------------|-------------------------------------------------|
| **Live Code Editor** | Редактор кода сториса                               | [README](src/addons/live-code-editor/README.md) |
| **Color Scheme**     | Переключатель цветовой схемы (light/dark) в тулбаре | [README](src/addons/color-scheme/README.md)     |
| **Storybook Theme**  | Переключатель темы интерфейса Storybook             | [README](src/addons/storybook-theme/README.md)  |
| **Source Button**    | Кнопка ссылки на исходный код сториса               | [README](src/addons/source-button/README.md)    |
