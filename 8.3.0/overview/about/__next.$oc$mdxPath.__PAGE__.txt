# Обзор

VKUI — это библиотека адаптивных React-компонентов, предназначенных для разработки веб-приложений.
Основана на одноименной [дизайн-системе ВКонтакте](https://www.figma.com/@vk).

</Overview>

## Преимущества

- **Адаптивность** — библиотека подстраивается под различные размеры экранов,
  обеспечивая удобный интерфейс на смартфонах, планшетах и компьютерах.

- **Мультиплатформенность** — интерфейсы VKUI практически идентичны дизайну других платформ,
  в которые они могут быть интегрированы. Благодаря способности "мимикрировать",
  библиотека может адаптироваться под iOS и android клиенты, сохраняя стиль дизайна,
  характерный для [vk.com](https://vk.com) и [m.vk.com](https://m.vk.com).

- **Разные цветовые схемы** — использование различных цветовых схем, включая темную тему,
  является одним из ключевых аспектов обеспечения доступности.

## Поддерживаемые браузеры

Для возможности использования нативной функциональности, каждый год увеличиваются версии
поддерживаемых браузеров на основе статистики команды Вконтакте. На данный момент версии следующие:

```Browserslist filename=".browserslistrc" file=<rootDir>/.browserslistrc
```

## Используемые полифилы

Компоненты библиотеки используют или могут использовать следующие полифилы:

- <Icon12ErrorCircleFill display="inline" /> [CSS logical properties and values](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Logical_properties_and_values) –  в CSS-бандле `@vkontakte/vkui/dist/vkui.css` при [базовой установке](/overview/install/#base-install), трансформируются в обычные свойства; при [установке ESNext версии](/overview/install/#esnext-install) решение о полифиле можно принять самому

## Релизный цикл

Для версионирования придерживаемся [semver](https://semver.org/). Временные рамки следующие:

- **Патч-версии** (x.y.Z) выпускаются по необходимости, при наличии исправлений;
- **Минорные версии** (x.Y.z) выпускаются с периодичностью один-два раза в месяц;
- **Мажорные версии** (X.y.z) с обратно несовместимыми изменениями выходят один раз в год.

обратитесь к разделу [Установка](/overview/install).

## Дизайн

На странице [VK в Figma Community](https://www.figma.com/@vk) доступны две библиотеки:

- **VKUI Common Library [Beta]** — основная библиотека компонентов для android, также используется для
  создания компонентов на десктопе;
- **VKUI iOS Library** — библиотеки компонентов для iOS c шаблонами готовых экранов;

Вы можете использовать эти открытые дизайн-библиотеки для реализации своих интерфейсов.
