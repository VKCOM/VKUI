## Использование

Компонент `FloatingArrow` следует использовать в случае реализации своего компонента всплывающей подсказки или
компонента онбординга.

`FloatingArrow` отвечает только визуальную составляющую, передавайте соответствующие параметры (свойство `coords`) для
корректного позиционирования компонента рядом с нужным элементом самостоятельно.
