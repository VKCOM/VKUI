# Mark

Компонент для визуального выделения фрагментов текста. Используется в поисковых результатах, цитатах и других сценариях, где требуется акцентировать внимание на определённых словах или фразах.

</Overview>

## Основное использование

Оберните текст, который нужно выделить:

```jsx
<Div>
  Что? <Mark>Где</Mark>? Когда?
  <br />
  <Mark>Где</Mark> эта улица, <Mark>где</Mark> этот дом?
</Div>
```

## Особенности

- Наследует все свойства HTML-элемента `<div>`
- Стандартное выделение через фон и цвет текста
- Поддерживает многострочное выделение
- Совместим с любым текстовым контентом
