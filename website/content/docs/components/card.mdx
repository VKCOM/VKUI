<Overview>

# Card

Компонент для создания визуально выделенных контейнеров с содержимым. Обеспечивает:

- Различные варианты стилевого оформления
- Семантически правильную разметку по умолчанию
- Гибкую интеграцию в разные типы интерфейсов

</Overview>

## Доступность (a11y)

- По умолчанию используется семантически правильный тег `<li>`
- Для одиночных карточек вне списков меняйте тег через `Component`
- Добавляйте ARIA-роли при необходимости
