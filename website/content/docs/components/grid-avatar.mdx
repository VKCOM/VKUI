<Overview>

# GridAvatar

Компонент для отображения нескольких аватаров в виде сетки. Поддерживает от 1 до 4 элементов.

</Overview>

## Основное использование

Передавайте массив изображений в свойство `src`:

```jsx
<GridAvatar src={['/user1.jpg', '/user2.jpg', '/user3.jpg', '/user4.jpg']} />
```

**Особенности отображения:**

- 1 элемент — занимает всю область
- 2 элемента — равные половины
- 3 элемента — один большой + два маленьких
- 4 элемента — равные четверти

## Добавление бейджа

Используйте подкомпонент `Badge` для индикаторов:

```jsx
<GridAvatar src={['/user1.jpg', '/user2.jpg']}>
  <GridAvatar.Badge background="stroke">
    <Icon20AddCircleFill />
  </GridAvatar.Badge>
</GridAvatar>
```

## Важные ограничения

- Минимум 1 элемент в массиве `src`
- Максимум 4 элемента (лишние игнорируются)

Пример с 3 элементами:

```jsx
<GridAvatar size={96} src={['/user1.jpg', '/user2.jpg', '/user3.jpg']} alt="Участники группы" />
```
