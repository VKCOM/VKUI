# CardGrid

Компонент-контейнер для создания адаптивных сеток из [Card](https://vkcom.github.io/VKUI/#/Card). Обеспечивает:

- Автоматическое выравнивание карточек
- Контроль пропорций элементов
- Настройку отступов
- Семантически правильную разметку по умолчанию

Связанные компоненты:

- Card

- ContentCard

</Overview>

## Размер карточек

Определяется свойством `size` - определяет пропорции карточек через соотношение высоты к ширине:

| Размер | Пропорции     | Типовое применение      |
| ------ | ------------- | ----------------------- |
| `s`    | 92% (≈1:1.08) | Квадратные элементы     |
| `m`    | 62% (≈1:1.61) | Горизонтальные карточки |
| `l`    | 30% (≈1:3.33) | Широкие баннеры         |

## Доступность (a11y)

- По умолчанию используется семантический тег `<ul>`
- Для одиночных сеток вне списков меняйте тег через `Component`
- Добавляйте ARIA-роли при необходимости
