# CardScroll

Компонент для создания горизонтально прокручиваемых списков [Card](https://vkcom.github.io/VKUI/#/Card). Обеспечивает:

- Контроль пропорций карточек
- Автоматическую прокрутку с кнопками-стрелками
- Гибкую настройку базовой разметки

Связанные компоненты:

- Card

- ContentCard

</Overview>

## Размер карточек

Свойство `size` определяет пропорции карточек через соотношение высоты к ширине:

| Размер  | Пропорции     | Особенности                   |
| ------- | ------------- | ----------------------------- |
| `s`     | 66% (≈1:1.5)  | Компактные элементы           |
| `m`     | 42% (≈1:2.38) | Универсальный размер          |
| `l`     | 29% (≈1:3.45) | Широкие элементы              |
| `false` | Авто          | Ширина определяется контентом |

## Управление видимостью стрелочек

В реализации компонента используется компонент [HorizontalScroll](https://vkcom.github.io/VKUI/#/HorizontalScroll). Поэтому для управления видимостью стрелочек можно использовать свойство `showArrows` компонента `HorizontalScroll`

```jsx
<CardScroll size="s" showArrows="always">
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
  <Card>
    <div style={{ paddingBottom: '66%' }} />
  </Card>
</CardScroll>
```

## Доступность (a11y)

- По умолчанию используется семантический список (`<ul>`)
- Для несписочных структур меняйте тег через `CardsListComponent`
