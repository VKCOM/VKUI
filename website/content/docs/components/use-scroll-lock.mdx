# useScrollLock

Хук для блокировки прокрутки страницы. Полезен при открытии модальных окон, всплывающих элементов и других элементов,
которые требуют ограничения взаимодействия с фоном.

</Overview>

## Пример использования

```jsx
const App = () => {
  const [opened, setOpened] = React.useState(false);

  // Блокируем скролл при открытии попапа
  useScrollLock(opened);

  return (
    <div>
      <h1>My Awesome App</h1>
      <label htmlFor="toggle">Toggle popout</label>
      <input
        id="toggle"
        type="checkbox"
        checked={opened}
        onChange={(e) => setOpened(e.target.checked)}
      />
      {opened && <PopoutWrapper>Some content</PopoutWrapper>}
    </div>
  );
};
```

## Требования

> Убедитесь, что ваше приложение обёрнуто в компонент `AppRoot`, так как именно через этот компонент:
>
> - Определяется контекст для управления прокруткой
> - Устанавливается уровень блокировки (полезно для вложенных модальных окон)

## Особенности

- Автоматически снимает блокировку при размонтировании компонента.
- Поддерживает вложенные блокировки (несколько модальных окон одновременно).
