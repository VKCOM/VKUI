<Overview>

# Accordion

Компонент для создания раскрывающихся/сворачивающихся блоков контента.

</Overview>

## Контролируемое состояние

Используйте `expanded` и `onChange` для ручного управления состоянием.

```jsx
function Example() {
  const [expanded, setExpanded] = React.useState();
  return (
    <Accordion expanded={expanded} onChange={setExpanded}>
      <Accordion.Summary iconPosition="before">Новый дизайн профиля</Accordion.Summary>
      <Accordion.Content>
        <Div>
          Внешний вид профиля ВКонтакте действительно обновился. К прежнему варианту вернуться уже
          не получится. В центре внимания нового дизайна — личность человека и его увлечения. Новый
          формат профиля особенно удобен для авторов контента и станет для них цифровой визиткой.
        </Div>
      </Accordion.Content>
    </Accordion>
  );
}

<Example />;
```

## Группа аккордеонов

Можно использовать компонент `Group` для отображения нескольких `Accordion`

```jsx
function Example() {
  const data = [
    {
      id: 1,
      title: 'Как сменить номер телефона?',
      detail:
        'Зайдите в раздел Безопасность и вход. Укажите новый номер телефона и подтвердите его.',
    },
    {
      id: 2,
      title: 'Как изменить пароль?',
      detail:
        'Для входа во многие сервисы экосистемы VK используется один пароль. Если вы беспокоитесь по поводу безопасности, этот пароль можно изменить.',
    },
    {
      id: 3,
      title: 'Как усилить защиту аккаунта?',
      detail:
        'Используйте сложный пароль и регулярно меняйте его. Обновить пароль можно в разделе Пароль.',
    },
  ];

  const [openId, setOpenId] = React.useState(2);

  return data.map(({ id, title, detail }) => (
    <Accordion
      key={id}
      expanded={openId === id}
      onChange={(e) => (e ? setOpenId(id) : setOpenId(null))}
    >
      <Accordion.Summary>{title}</Accordion.Summary>
      <Accordion.Content>
        <Div style={infoStyle}>{detail}</Div>
      </Accordion.Content>
    </Accordion>
  ));
}

<Example />;
```

## Свои иконки

Используйте свойства `ExpandIcon` и `CollapseIcon`.

```jsx
function Example() {
  const [expanded, setExpanded] = React.useState();
  return (
    <Accordion expanded={expanded} onChange={setExpanded}>
      <Accordion.Summary
        iconPosition="before"
        ExpandIcon={Icon28AddCircleOutline}
        CollapseIcon={Icon28RemoveCircleOutline}
      >
        Новый дизайн профиля
      </Accordion.Summary>
      <Accordion.Content>
        <Div>
          Внешний вид профиля ВКонтакте действительно обновился. К прежнему варианту вернуться уже
          не получится. В центре внимания нового дизайна — личность человека и его увлечения. Новый
          формат профиля особенно удобен для авторов контента и станет для них цифровой визиткой.
        </Div>
      </Accordion.Content>
    </Accordion>
  );
}

<Example />;
```

## Анимация

- Контент плавно раскрывается/сворачивается
- Автоматически рассчитывает высоту содержимого
- Поддерживает прерывание анимации при быстром клике
