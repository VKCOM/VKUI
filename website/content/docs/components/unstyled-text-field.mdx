# UnstyledTextField

Компонент сбрасывает [User-agent stylesheets](https://www.geeksforgeeks.org/what-is-a-user-agent-stylesheet/) полей ввода.

</Overview>

Используйте данный компонент, если вы разрабатываете своё поле вода и хотите сбросить все браузерные стили, которые
применяются по умолчанию к таким компонентам как `<input>` и `<textarea>`.

```jsx
<UnstyledTextField placeholder="Поле без стилей" />
```

## Тэг компонента

Используйте свойства `as`, которое позволяет изменить тэг, который рендерится по умолчанию (`<textarea>`).
