# UnstyledTextField

Компонент сбрасывает [User-agent stylesheets](https://www.geeksforgeeks.org/what-is-a-user-agent-stylesheet/)
таких полей ввода, как `<input>` и `<textarea>`.

</Overview>

Используйте данный компонент, если вы разрабатываете своё поле вода и хотите сбросить все браузерные стили, которые
применяются по умолчанию к таким компонентам,как `<input>` и `<textarea>`.

```jsx
<>
  <UnstyledTextField as="input" placeholder="No styles input" />
  <input placeholder="Browser styled input" />
</>
```

## Обязательные свойства

### Тэг компонента

Используйте свойства `as`, которое позволяет задать тип поля воода (`"textarea"` или `"input"`).
