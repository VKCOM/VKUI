# SelectMimicry

Компонент, который визуально имитирует компонент выбора из списка.
Используется в случаях, когда нужно создать элемент, похожий на `Select`,
но с пользовательским поведением при нажатии.

</Overview>

```jsx
<SelectMimicry placeholder="Выберите значение">Значение</SelectMimicry>
```

## Доступность (a11y)

Компонент поддерживает все стандартные `aria`-атрибуты, если вам необходимо переопределить
стандартное поведение.
