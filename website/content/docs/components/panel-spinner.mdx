<Overview>

# PanelSpinner

Компонент-обёртка для индикатора загрузки, оптимизированный для использования внутри панелей. Автоматически добавляет стандартные отступы и центрирование.

Связанные компоненты:

- Spinner
- ScreenSpinner

</Overview>
## Основное использование

```jsx
<Panel>
  <PanelHeader>Профиль</PanelHeader>
  {isLoading ? <PanelSpinner /> : <UserContent />}
</Panel>
```

## Свойства

### Наследует все свойства [`Spinner`](#/Spinner):

- `size` — размер индикатора
- `disableAnimation` — отключение анимации
- `noColor` — наследование цвета

### Высота компонента:

Задается с помощью свойства `height`:

```jsx
<PanelSpinner height={300} />
```

## Когда использовать

- При загрузке контента внутри панели
- Когда нужен центрированный спиннер с отступами
- Для полностраничной загрузки (в сочетании с кастомной высотой)

## Альтернативы

- [`Spinner`](#/Spinner) — для встроенной загрузки внутри элементов
- [`ScreenSpinner`](#/ScreenSpinner) — для модальной загрузки поверх интерфейса

## Доступность (a11y)

Следуюет придерживаться тех же правил, что и для компонента [`Spinner`](#/Spinner)
