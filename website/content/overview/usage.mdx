## Навигационные слои [#navigation]

Каждое приложение так или иначе состоит из нескольких экранов с информацией. В VKUI есть специальные компоненты для реализации
этих экранов и навигации между ними. С такими компонентами вы подробней ознакомитесь далее на странице [Навигация](/overview/navigation),
а сейчас лишь посмотрим на простой пример как это вообще выглядит.

{/* TODO: для этого кода нужно превью, код должен быть скрыт по умолчанию */}

```jsx filename="src/App.tsx"
import { useState } from 'react';
import {
  SplitLayout,
  SplitCol,
  View,
  Panel,
  PanelHeader,
  Group,
  CellButton,
} from '@vkontakte/vkui';

export default function App() {
  const [activePanel, setActivePanel] = useState('panel-1');

  return (
    <SplitLayout header={<PanelHeader delimiter="none" />}>
      <SplitCol width="100%" stretchedOnMobile autoSpaced>
        <View activePanel={activePanel}>
          <Panel id="panel-1">
            <PanelHeader>panel-1</PanelHeader>
            <Group>
              <CellButton onClick={() => setActivePanel('panel-2')}>Перейти к panel-2</CellButton>
            </Group>
          </Panel>
          <Panel id="panel-2">
            <PanelHeader>panel-2</PanelHeader>
            <Group>
              <CellButton onClick={() => setActivePanel('panel-1')}>Вернуться к panel-1</CellButton>
            </Group>
          </Panel>
        </View>
      </SplitCol>
    </SplitLayout>
  );
}
```

## Многоколоночный интерфейс [#multiple-columns]

Если на мобильных устройствах приходится ограничивать количество элементов в одной области, то на настольных, напротив, можно
размещать их значительно больше. Для этого интерфейс, как правило, делится на колонки. Для этого в библиотеке есть компонент
[`SplitLayout`](/components/split-layout) с подкомпонентом [`SplitCol`](/components/split-layout#split-col). Детали будут также
на странице [Навигация](/overview/navigation).

Приведём адаптивный пример работы `SplitLayout`.

{/* TODO: для этого кода нужно превью, код должен быть скрыт по умолчанию */}

```jsx filename="src/App.tsx"
import { SplitLayout, SplitCol, useAdaptivityConditionalRender } from '@vkontakte/vkui';

export default function App() {
  const { viewWidth } = useAdaptivityConditionalRender();

  return (
    <SplitLayout>
      {viewWidth.tabletPlus && (
        <SplitCol fixed width={280} maxWidth={280} className={viewWidth.tabletPlus.className}>
          <div
            style={{
              padding: 12,
              height: 80,
              color: 'white',
              backgroundColor: 'tomato',
            }}
          >
            Фиксированная колонка слева
          </div>
        </SplitCol>
      )}
      <SplitCol>
        <div
          style={{
            padding: 12,
            height: 'calc(100% - 24px)',
            color: 'white',
            backgroundColor: 'steelblue',
          }}
        >
          Колонка справа
          {viewWidth.tabletMinus && (
            <p className={viewWidth.tabletMinus.className}>
              ⚠️ колонка слева спрятана по условию адаптивности
            </p>
          )}
        </div>
      </SplitCol>
    </SplitLayout>
  );
}
```
