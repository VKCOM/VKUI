# Обзор

VKUI — это библиотека адаптивных React-компонентов, предназначенных для разработки веб-приложений.
Основана на одноименной [дизайн-системе ВКонтакте](https://www.figma.com/@vk).

</Overview>

## Преимущества

- **Адаптивность** — библиотека подстраивается под размер экрана: удобно на смартфонах, планшетах и компьютерах.

- **Мультиплатформенность** — VKUI «мимикрирует» под iOS и Android, сохраняя стиль [vk.com](https://vk.com) и [m.vk.com](https://m.vk.com). Интерфейсы выглядят естественно в любом окружении.

- **Цветовые схемы** — есть светлая и тёмная темы, что важно для доступности.

## Поддерживаемые браузеры

Каждый год мы повышаем минимальные версии браузеров на основе статистики ВКонтакте. Текущие версии:

```Browserslist filename=".browserslistrc" file=<rootDir>/.browserslistrc
```

## Используемые полифилы

Компоненты используют или могут использовать следующие полифилы:

- <Icon12ErrorCircleFill display="inline" /> [CSS logical properties and values](https://developer.mozilla.org/en-US/docs/Web/CSS/Guides/Logical_properties_and_values) — в CSS-бандле `@vkontakte/vkui/dist/vkui.css` при [базовой установке](/overview/install/#base-install) трансформируются в обычные свойства; при [установке ESNext-версии](/overview/install/#esnext-install) решение о полифиле вы принимаете сами.

## Релизный цикл

Версионирование — по [semver](https://semver.org/). Сроки выпуска:

- **Патчи** (x.y.Z) — по мере исправлений;
- **Миноры** (x.Y.z) — один-два раза в месяц;
- **Мажоры** (X.y.z) с обратно несовместимыми изменениями — раз в год.

## Дизайн

На [странице VK в Figma Community](https://www.figma.com/@vk) доступны две библиотеки:

- **VKUI Common Library [Beta]** — основная библиотека компонентов для Android, также используется для десктопа;
- **VKUI iOS Library** — компоненты для iOS с шаблонами готовых экранов;

Это открытые дизайн-библиотеки — используйте их для своих интерфейсов.
