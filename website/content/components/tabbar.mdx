# Tabbar

Компонент для организации навигации внутри `Epic`, принимает компонент `TabbarItem` в качестве дочерних элементов.
Используется на мобильных устройствах и планшетах.

Связанные компоненты:

- [`Epic`](/components/epic)
- [`TabbarItem`](/components/tabbar-item)

## `TabbarItem`

Передавайте компоненты `TabbarItem` в свойство `children` для создания навигации по разделам вашего приложения.

```jsx
<Tabbar>
  <TabbarItem label="Новости">
    <Icon28NewsfeedOutline />
  </TabbarItem>
  <TabbarItem label="Профиль" selected>
    <Icon28UserCircleOutline />
  </TabbarItem>
</Tabbar>
```

### Выравнивание внутри `TabbarItem`

В случае, если в `Tabbar` передан один или два элемента `TabbarItem`, контент внутри этих элементов располагается по горизонтали,
в остальных случаях контент располагается вертикально.

С помощью свойства `mode` у `Tabbar` можно переопределить это поведение.

- `"vertical"` — задаёт всегда вертикальное расположение;
- `"horizontal"` — задаёт всегда горизонтальное расположение.

<Playground Wrapper={FixedLayoutWrapper} style={{ height: 60 }}>
  ```jsx
  <Tabbar mode="vertical">
    <TabbarItem label="Новости">
      <Icon28NewsfeedOutline />
    </TabbarItem>
    <TabbarItem label="Профиль" selected>
      <Icon28UserCircleOutline />
    </TabbarItem>
  </Tabbar>
  ```
</Playground>

## Внешний вид

Свойство `plain` позволяет убрать границы компонента (на android-платформе это тень, на iOS-платформе обводка).

Используйте данное свойство в случае, если выше `Tabbar` располагается какой-то фиксированный контент
(например, `FixedLayout` c `vertical="bottom"`).

<Playground Wrapper={FixedLayoutWrapper} style={{ height: 60 }}>
  ```jsx
  <Tabbar mode="vertical" plain>
    <TabbarItem label="Новости">
      <Icon28NewsfeedOutline />
    </TabbarItem>
    <TabbarItem label="Профиль" selected>
      <Icon28UserCircleOutline />
    </TabbarItem>
  </Tabbar>
  ```
</Playground>
