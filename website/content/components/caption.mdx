# Caption [tag:component]

Компонент для отображения мелких подписей — аннотаций к изображениям, временных меток,
вспомогательных аннотаций и другого вторичного текста минимального размера.

</Overview>

{/* @example-description: Базовое использование компонента `Caption`. */}
<Playground>
  ```jsx
  <Caption>Мелкая подпись</Caption>
  ```
</Playground>

## Базовые свойства

Компонент поддерживает все [базовые свойства типографики](/components/typography#base-props):
`weight`, `useAccentWeight`, `normalize`, `Component`, `inline`, `align`.

## Уникальные свойства

### level

Определяет иерархический уровень подписи. Доступны значения `'1' | '2' | '3'`, где `'1'` — самый крупный.
По умолчанию `level="1"`.

{/* @example-description: Три уровня Caption для сравнения размеров. */}
<Playground>
  ```jsx
  <Flex direction="column">
    <Caption level="1">Caption уровень 1</Caption>
    <Caption level="2">Caption уровень 2</Caption>
    <Caption level="3">Caption уровень 3</Caption>
  </Flex>
  ```
</Playground>

### caps

Трансформирует все буквы в заглавные (`text-transform: uppercase`). Удобно для меток и аббревиатур.
Можно сочетать с любым уровнем `level`.

{/* @example-description: Использование `caps` в `Caption` для отображения текста в верхнем регистре. */}
<Playground>
  ```jsx
  <Caption caps>Подпись в верхнем регистре</Caption>
  ```
</Playground>
