# Title [tag:component]

Компонент для отображения заголовков страниц и крупных секций интерфейса.

</Overview>

{/* @example-description: Заголовок `Title` первого уровня по умолчанию. */}
<Playground>
  ```jsx
  <Title>Заголовок</Title>
  ```
</Playground>

## Базовые свойства

Компонент поддерживает все [базовые свойства типографики](/components/typography#base-props):
`weight`, `useAccentWeight`, `normalize`, `Component`, `inline`, `align`.

## Уникальные свойства

### level

Определяет иерархический уровень заголовка. Доступны значения `'1' | '2' | '3'`, где `'1'` — самый крупный.
По умолчанию `level="1"`.

{/* @example-description: Три уровня Title для сравнения размеров. */}
<Playground>
  ```jsx
  <Flex direction="column">
    <Title level="1">Title уровень 1</Title>
    <Title level="2">Title уровень 2</Title>
    <Title level="3">Title уровень 3</Title>
  </Flex>
  ```
</Playground>
