# SplitLayout

Компонент-контейнер для реализации интерфейса с многоколоночной структурой.
За реализацию колонки отвечает `SplitCol`.

</Overview>

## Выравнивание

За выравнивание контента по горизонтали отвечает свойство `center`.

- `true` — весь контент выравнивается по центру;
- `false` — выравнивание не применяется.

## Шапка

Для реализации сквозной шапки между колонками используется свойство `header`.

> Если вы реализуете интерфейс для платформы, отличной от `vkcom`,
> то в это свойство обязательно нужно передавать `<PanelHeader delimiter="none" />`.

```jsx
const platform = usePlatform();
const hasHeader = platform !== 'vkcom';

<SplitLayout header={hasHeader && <PanelHeader delimiter="none" />}>
  {/* остальной контент */}
</SplitLayout>;
```

## Модальные и всплывающие окна

До версии `VKUI v7` использовать модальные и всплывающие окна без компонента `SplitLayout` было нельзя.
Необходимо было передавать `ModalRoot` со списком модальных окон в свойство `modal`,
а всплывающие элементы (`Alert`, `ActionSheet` и `ScreenSpinner`) в свойство `popout`.

На данный момент вы можете определять модальные и всплывающие окна в любом месте своего приложения в пределах `AppRoot`,
а свойства `modal` и `popout` отмечены устаревшими и будут удалены в `VKUI v8`.
