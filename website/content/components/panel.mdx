<Overview>
# Panel

Компонент-контейнер, представляющий собой навигационный слой, содержащий основной контент страницы.

</Overview>

## Уникальный идентификатор

Уникальный идентификатор для `Panel` можно задать либо с помощью свойства `nav`, либо `id`.
Обязательно указывайте идентификатор, когда используете компонент внутри `View`.

## Визуальное оформление

### Тип отображения

Задаётся свойством `mode`.

- `"card"` — карточный дизайн с тенями и скруглениями;
- `"plain"` — плоский дизайн без теней и скруглений.

### Центрирование контента

Задаётся свойством `centered`.

### Фон страницы

Задаётся свойством `disableBackground`.

- `true` — задаёт прозрачный цвет фона;
- `false` — стандартный цвет фона.
