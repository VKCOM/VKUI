# AdaptivityProvider

[Обязательный компонент](/overview/usage#required-wrapper), оборачивающий всё приложение. Используется для управления отображением
компонентов в зависимости [параметров адаптивности](/overview/adaptivity#adaptivity-props).

Связанные страницы:

- [Адаптивность](/overview/adaptivity)

</Overview>

## Автоматическое определение свойств

`AdaptivityProvider` автоматически определяет некоторые свойства на основе других:

- если `sizeX` не задан, но задан `viewWidth`, то `sizeX` будет вычислен автоматически;
- если `sizeY` не задан, то он будет определен на основе `viewWidth`, `viewHeight` и `hasPointer`.

## Хук useAdaptivity

Хук возвращает текущие "сырых" значения адаптивности.

```jsx
import { useAdaptivity } from '@vkontakte/vkui';

export function App() {
  const { sizeX, sizeY, viewWidth, viewHeight, hasPointer } = useAdaptivity();

  return (
    <pre>
      sizeX: {sizeX}, sizeY: {sizeY}, viewWidth: {viewWidth}, viewHeight: {viewHeight}, hasPointer:{' '}
      {hasPointer}
    </pre>
  );
}
```

<Callout>
  Для создание адаптивных интерфейсов лучше использовать более гибкие утилиты (см. [Адаптивность |
  Утилиты для создания адаптивных интерфейсов](/overview/adaptivity#utils)).
</Callout>
