<Overview>

# AdaptivityProvider

[Обязательный компонент](/overview/usage#required-wrapper), оборачивающий всё приложение и позволяющий задавать или переопределять данные об адаптивности.
Используется для управления отображением компонентов в зависимости от размера экрана,
типа устройства и других параметров адаптивности.

</Overview>

## Автоматическое определение свойств

`AdaptivityProvider` автоматически определяет некоторые свойства на основе других:

- если `sizeX` не задан, но задан `viewWidth`, то `sizeX` будет вычислен автоматически;
- если `sizeY` не задан, то он будет определен на основе `viewWidth`, `viewHeight` и `hasPointer`.

## Переопределение адаптивности

Для того, чтобы переопределить тот или иной параметр адаптивности, оберните необходимую часть приложения в
`AdaptivityProvider` и передайте нужные свойства. Все нижележащие компоненты получат изменённое отображение в
соответствии с переданными значениями.

> Обратите внимание, что изменение параметров адаптивности может _негативно_ сказаться на пользовании вашим приложением,
> поэтому старайтесь этого избегать.

Наиболее частные параметры, которые вы можете захотеть изменить - это `sizeX` и `sizeY`,
они влияют на размеры компонентов. На мобильных устройства компоненты _намеренно_ сделаны больше,
чтобы было удобнее нажимать пальцем.

```jsx
<AdaptivityProvider sizeY="compact">
  <Button>Пальцем не попадёшь</Button>
</AdaptivityProvider>
```

## Хук useAdaptivity

Для получения текущих значений адаптивности используйте хук `useAdaptivity`:

```jsx
import { useAdaptivity } from '@vkontakte/vkui';
```

```jsx
const { sizeX, sizeY, viewWidth, viewHeight, hasPointer, hasHover } = useAdaptivity();
```
