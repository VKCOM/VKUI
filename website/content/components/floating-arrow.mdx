## Использование

`FloatingArrow` используйте при реализации собственного компонента всплывающей подсказки или онбординга.

`FloatingArrow` отвечает только за визуальную составляющую. Для корректного позиционирования рядом с нужным элементом передавайте соответствующие параметры (свойство `coords`) самостоятельно.
