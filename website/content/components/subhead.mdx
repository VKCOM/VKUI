# Subhead [tag:component]

Компонент для отображения подзаголовков второго уровня. Используется для разделения контента внутри секций,
а также как акцентные подписи меньшего размера, чем [`Headline`](/components/headline).

</Overview>

{/* @example-description: Базовое использование компонента `Subhead`. */}
<Playground>
  ```jsx
  <Subhead>Подзаголовок второго уровня</Subhead>
  ```
</Playground>

## Базовые свойства

Компонент поддерживает все [базовые свойства типографики](/components/typography#base-props):
`weight`, `useAccentWeight`, `normalize`, `Component`, `inline`, `align`.

`Subhead` не имеет уникальных свойств — вся кастомизация осуществляется через базовые свойства типографики.
