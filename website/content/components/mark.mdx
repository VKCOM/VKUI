# Mark

Компонент для визуального выделения фрагментов текста. Используется в поисковых результатах,
цитатах и других сценариях, где требуется акцентировать внимание на определённых словах или фразах.

</Overview>

<Playground>
  ```jsx
  <Text>
    Что? <Mark>Где</Mark>? Когда?
  </Text>
  ```
</Playground>

## Особенности

- Наследует все свойства HTML-элемента `<div>`.
- Стандартное выделение через фон, цвет текста наследуется.
- Поддерживает многострочное выделение.
- Совместим с любым текстовым контентом.

## Доступность (a11y)

Используется стандартный тэг `mark` для семантического обозначения выделенного текста.

## Рекомендации

- Не используйте для интерактивных элементов.
- Сочетайте с текстовыми компонентами ([`Text`](/components/typograhy), [`Headline`](/components/typograhy), [`Paragraph`](/components/typograhy)).
