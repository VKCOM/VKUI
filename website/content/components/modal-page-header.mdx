# ModalPageHeader

Шапка в модальных страницах.
Имеет очень похожую логику на [`PanelHeader`](/components/panel-header): два свойства `before` и `after` для иконок и
название модальной страницы, передаваемое как `children`.

</Overview>

## Требования по расположению кнопок

- На Android **слева** может быть [`PanelHeaderClose`](/components/panel-header-button#panelheaderclose).
- На iOS **справа** должна быть кнопка закрытия. Это может быть либо [`PanelHeaderButton`](/components/panel-header-button) с иконкой `<Icon24Dismiss />`, либо [`PanelHeaderClose`](/components/panel-header-button#panelheaderclose) или [`PanelHeaderSubmit`](/components/panel-header-button#panelheadersubmit).
- Если произойдёт навигация вперёд внутри модального окна, то **слева** в шапке останется **только** кнопка назад.

Также на Android могут быть какие-то дополнительные кнопки-иконки справа, а на iOS – слева.

```jsx
const platform = usePlatform();

return (
  <ModalPageHeader
    before={
      <Fragment>
        {(platform === 'android' || platform === 'vkcom') && (
          <PanelHeaderButton onClick={this.backModal}>
            <Icon24Cancel />
          </PanelHeaderButton>
        )}
      </Fragment>
    }
    after={
      <Fragment>
        {(platform === 'android' || platform === 'vkcom') && (
          <PanelHeaderButton onClick={this.backModal}>
            <Icon24Done />
          </PanelHeaderButton>
        )}
        {platform === 'ios' && (
          <PanelHeaderButton onClick={this.backModal}>Готово</PanelHeaderButton>
        )}
      </Fragment>
    }
  >
    Заголовок модальной страницы
  </ModalPageHeader>
);
```

Пример использования показан в компоненте `ModalRoot`.
