# Headline [tag:component]

Компонент для отображения подзаголовков и акцентного текста внутри секций интерфейса.

</Overview>

{/* @example-description: Подзаголовок `Headline` первого уровня по умолчанию. */}
<Playground>
  ```jsx
  <Headline>Подзаголовок</Headline>
  ```
</Playground>

## Базовые свойства

Компонент поддерживает все [базовые свойства типографики](/components/typography#base-props):
`weight`, `useAccentWeight`, `normalize`, `Component`, `inline`, `align`.

## Уникальные свойства

### level

Определяет иерархический уровень подзаголовка. Доступны значения `'1' | '2'`, где `'1'` — более крупный.
По умолчанию `level="1"`.

{/* @example-description: Два уровня Headline для сравнения размеров. */}
<Playground>
  ```jsx
  <Flex direction="column">
    <Headline level="1">Headline уровень 1</Headline>
    <Headline level="2">Headline уровень 2</Headline>
  </Flex>
  ```
</Playground>
